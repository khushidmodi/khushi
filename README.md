# Khushi Modi Portfolio

Personal portfolio website for showcasing my background, resume, selected work, awards, and contact information.

Live site: [https://khushidmodi.github.io/khushi/](https://khushidmodi.github.io/khushi/)

## Overview

This is a static HTML, CSS, and JavaScript portfolio deployed with GitHub Pages. The site includes:

- An About Me section with a personal statement and headshot
- A downloadable resume
- Selected projects and papers with custom images and external/download links
- Awards and recognitions
- Contact information and a dedicated contact page

## Project Structure

- `index.html` - Main portfolio page
- `contact.html` - Dedicated contact page
- `styles.css` - Site styling, layout, responsive behavior, and card/image treatments
- `script.js` - Small JavaScript helper for the footer year
- `assets/` - Headshot and project images
- `resume/` - Downloadable resume PDF
- `work-samples/` - Downloadable work sample files, including the data science paper

## Work Samples

The selected projects and papers section currently includes:

- Data Science Study - downloadable PDF
- Bus App Prototype - external prototype link
- Password Management System - external project link

## Local Preview

Open `index.html` in a browser to view the portfolio locally.

Because this is a static site, no build step or package installation is required.
